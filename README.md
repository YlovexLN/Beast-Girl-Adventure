<h1 align="center">Mod-UpdateScript</h1>

### generate-mod-list.bat

通过 CurseForge API 获取指定的 mod 信息，包括 mod 名称、版本号、下载地址，

并将这些信息组成 json 格式的 mod-list

最后再将mod-list上传到GitHub方便于本地的mod-list对比。

### update-mods.bat

检测远程仓库的mod-list与本地mod-list是否不一样

从而找到对应的mod实现mod更新。
